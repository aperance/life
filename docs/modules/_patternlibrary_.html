Module "patternLibrary"

Pattern Library is a collection of well known patterns provided to the user. These patterns are imported from ".rle" data files created by the game of life community. These files are located in the "patterns" folder of this repository. An associated dropdown menu is generated listing all available patterns for the user to select from.
