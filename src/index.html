Conway's Game of Life

This is an implementation of Conway's Game of Life written in Javascript and Rust (WebAssembly). The game universe is 5000 cells by 5000 cells, with the edges wrapping around.

How to Play

The initial state of the game (generation 0) can be set through two modes:

Toggle Cells - Click on an individual cell to toggle between alive and dead states.

Pattern Library - Choose from the included collection of common patterns. The selected pattern can be placed one or more times by clicking on the desired locations.

Once the initial state is set press Play to begin. Use the Speed slider to adjust the game speed. Editing is disabled once the game has started. Press Reset to start over.

Pan across the game area by using the arrow keys on your keyboard or by clicking and dragging on the game area.

Adjust the zoom level using the zoom slider or by scrolling using your mouse or touchpad.

Use Sun/Moon button to switch beteen light and dark modes.
